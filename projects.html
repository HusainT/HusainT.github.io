Projects

Algorithmic Trading Stock Classifier

I created an Algorithmic trading classifier in Python. It extracts stock price data from Yahoo finance and makes buy/sell decisions based on training from historical data. The goal is to have the return from the algorithmic trading algorithm exceed the expected return from buying and holding.

Alien Attack Arcade Game

A group of developers and I created Space Invaders and a custom Alien invasion survival arcade game. We used object-oriented programing concepts and wrote the game in C++ in the Arduino IDE. The game was played using an LED screen and controls connected to the Arduino through a breadboard. Our project won the grand prize at the end of the year EECS 183 final project showcase!
